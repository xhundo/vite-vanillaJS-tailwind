**Getting started**

 ```bash
 $ git clone https://github.com/xhundo/vite-vanillaJS-tailwind.git
 ```
 
 
 **Install packages via NPM**
 ```bash
 $ npm install
 # or `npm i`
 ```
 
 
**Develop**
 ```bash
$ npm run dev
$ npm run watch # `tailwind build` 
 ```
